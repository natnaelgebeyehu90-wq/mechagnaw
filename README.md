# 🚀 Mechagnaw

A fast, modern YouTube video & audio downloader built with Node.js, Express, Socket.IO and yt-dlp.

---

## ✨ Features

- 📹 Download YouTube videos
- 🎵 Download MP3 audio
- ⚡ Real-time download progress
- 🌙 Dark Mode
- 🌍 Multi-language support
- 📱 Responsive design
- 📋 One-click Paste button
- ⚡ Fast downloads powered by yt-dlp

---

## Technologies

- Node.js
- Express
- Socket.IO
- yt-dlp
- HTML
- CSS
- JavaScript

---

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/mechagnaw.git

cd mechagnaw

npm install

node server.js
```

Open

```
http://localhost:3000
```
